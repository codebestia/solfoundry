# Changelog

## v1.0.0 — 2024

- Initial SDK release with `BountyClient`, `EscrowClient`, `ContributorClient`
- `GitHubClient` for bounty issue integration
- `EventSubscriber` for real-time WebSocket events
- Solana helpers: PDA derivation, SPL token balance, tx building
- Typed error hierarchy with automatic retry and rate limiting
- `@solfoundry/cli` with `bounties`, `status`, `profile`, `verify` commands
- VitePress documentation site with 11 examples
- TypeDoc-generated API reference
